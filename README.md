# Student Course Management System

## 📌 Overview
This project represents a relational database system designed to manage students, courses, and their enrollments. It demonstrates core database design principles and SQL development techniques, including schema creation, data manipulation, and advanced querying.

The project was developed as part of a Master's degree in Computer Science.

---

## 🧱 Database Structure
The database is designed using a normalized relational model and includes the following core entities:

- Students
- Courses
- Enrollments

An Entity-Relationship (ER) diagram is included to illustrate the structure and relationships between tables.

---

## ⚙️ Features
- Creation of database schema using SQL
- Sample data population
- Full CRUD operations (Create, Read, Update, Delete)
- Stored procedures for reusable logic
- SQL views for simplified data access
- Structured and modular SQL scripts

---

## 📂 Project Structure
- `create_tables.sql` – Defines all database tables
- `insert_sample_data.sql` – Inserts test data
- `crud_operations.sql` – Contains CRUD queries
- `stored_procedures.sql` – Defines stored procedures
- `views.sql` – Contains SQL views
- `ER diagram.pdf` – Database design diagram
- `README.md` – Project documentation

---

## 🚀 How to Use
1. Create a new database in your SQL environment (MySQL / SQL Server).
2. Run the `create_tables.sql` script.
3. Execute `insert_sample_data.sql` to populate the database.
4. Use CRUD scripts and stored procedures to interact with the data.
5. Query the views for simplified data retrieval.
